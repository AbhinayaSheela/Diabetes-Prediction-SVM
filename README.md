# Diabetes-Prediction-SVM

##  Project Overview
This project uses a **Support Vector Machine (SVM)** model to predict whether a person has diabetes based on medical features. The model is trained on the Pima Indians Diabetes dataset and deployed using **Streamlit** for an interactive web application.


##  Dataset Features

- Pregnancies  
- Glucose  
- BloodPressure  
- SkinThickness  
- Insulin  
- BMI  
- DiabetesPedigreeFunction  
- Age  
- Outcome (Target)

###  Target:
- 0 → No Diabetes  
- 1 → Diabetes  


##  Technologies Used

- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Support Vector Machine (SVM)  
- Streamlit  
- Matplotlib  


##  Machine Learning Model

- Algorithm: SVM (Support Vector Machine)  
- Kernel: Linear  
- Scaling: StandardScaler  
- Problem Type: Binary Classification
